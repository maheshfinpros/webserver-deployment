GitHub Webhook Setup for Jenkins

  1. Configure Jenkins:
    • Go to Jenkins dashboard.
    • Navigate to Manage Jenkins > Configure System.
    • Under the "GitHub" section, ensure the API URL and credentials are correctly set.
  2. Set up GitHub Webhook:
    • Open your GitHub repository.
    • Go to Settings > Webhooks.
    • Click Add webhook.
    • In the "Payload URL" field, enter: http://52.66.24.205:8080/github-webhook/
    • Set the "Content type" to application/json.
    • Choose the events to trigger the webhook. The recommended option is Just the push event.
    • Click Add webhook.
